## Features

- **Poll Mode** - Monitor commands at configurable intervals
- **Log Capture** - Persist output to temp files for later analysis
- **AI Summarization** - Get concise summaries of command output
- **Pattern Matching** - Detect success/error conditions automatically
- **Post-Command Hooks** - Run follow-up commands on success or failure
