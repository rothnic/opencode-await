## Features

<CardGrid>
  <Card title="Blocking Execution" icon="rocket">
    Agent session pauses until command completes. No polling loops, no wasted API calls.
  </Card>
  <Card title="Concise Output" icon="document">
    Returns clean success/failure status instead of raw terminal dumps.
  </Card>
  <Card title="Simple Tool Calls" icon="pencil">
    One intuitive function call replaces complex shell command construction.
  </Card>
  <Card title="AI Summarization" icon="star">
