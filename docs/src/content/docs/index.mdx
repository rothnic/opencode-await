## Quick Install

```json
// opencode.json
{
  "plugin": ["opencode-await"]
}
```

## Features

<CardGrid>
  <Card title="Command Execution" icon="terminal">
    Run any shell command with configurable timeout up to 30 minutes.
  </Card>
  <Card title="Pattern Matching" icon="magnifier">
    Detect success or failure via regex patterns in output.
  </Card>
  <Card title="Poll Mode" icon="refresh">
    Periodically execute commands to check status.
  </Card>
  <Card title="AI Summarization" icon="star">
    Get AI-powered summaries of verbose command output.
  </Card>
</CardGrid>
