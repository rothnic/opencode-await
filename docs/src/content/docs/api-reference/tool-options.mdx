## await_command Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `command` | string | required | Shell command to execute |
| `maxDuration` | number | 300 | Maximum duration in seconds |
| `successPattern` | string | - | Regex for success detection |
| `errorPattern` | string | - | Regex for error detection |
| `pollMode` | object | - | Polling configuration |
| `persistLogs` | boolean | false | Save output to temp file |
| `logPath` | string | - | Custom log file path |
| `summarize` | object | - | AI summarization config |
| `onSuccess` | string | - | Command to run on success |
| `onFailure` | string | - | Command to run on failure |

## Poll Mode Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `enabled` | boolean | required | Enable polling |
| `interval` | number | 5 | Seconds between polls |
| `appendOutput` | boolean | true | Append vs replace output |

## Summarize Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `enabled` | boolean | required | Enable summarization |
| `model` | string | gpt-5-mini | LLM model to use |
