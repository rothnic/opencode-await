## Log Capture

Enable log persistence to save command output to a temporary file:

```typescript
await_command({
  command: "npm run long-task",
  persistLogs: true
})
```

The result will include a `logPath` field pointing to the saved output.

## Custom Log Path

Optionally specify a custom path:

```typescript
await_command({
  command: "npm run build",
  persistLogs: true,
  logPath: "/tmp/my-build.log"
})
```
