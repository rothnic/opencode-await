## Poll Mode

Poll mode executes a command repeatedly at specified intervals until a pattern matches or timeout occurs.

```typescript
await_command({
  command: "kubectl get pods",
  maxDuration: 600,
  successPattern: "Running",
  pollMode: {
    enabled: true,
    interval: 5  // Check every 5 seconds
  }
})
```

## Options

- `enabled`: Enable polling mode
- `interval`: Seconds between polls (1-60)
- `appendOutput`: Append vs replace output each poll
