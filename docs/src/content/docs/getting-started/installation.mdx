## Installation

Add `opencode-await` to your project's `opencode.json`:

```json
{
  "plugin": ["opencode-await"]
}
```

That's it! OpenCode will automatically install the plugin on next startup.

## Requirements

- OpenCode v1.0.0 or later
- Bun runtime (for plugin execution)
